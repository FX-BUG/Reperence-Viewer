# RPview

GIF, PNG, JPG, MP4 파일을 캔버스에 자유롭게 배치해 참고 이미지/영상으로 활용하는 레퍼런스 뷰어입니다.

## 설치 방법

### 요구 사항

- Windows 10 / 11 (64bit)
- 인터넷 연결 (최초 설치 시)

<br>

### 설치

**1. 저장소 다운로드**

오른쪽 상단 `Code` → `Download ZIP` 을 클릭하거나, 아래 명령어로 클론합니다.

```bash
git clone https://github.com/your-username/RPview.git
```

**2. 설치 스크립트 실행**

압축 해제 후 폴더 안의 `install-bat.bat` 를 더블클릭합니다.

```
install-bat.bat
```

Python 3.11과 필요한 패키지(PyQt5, OpenCV 등)가 자동으로 설치됩니다.

**3. 실행**

설치 완료 후 `review-bat.bat` 를 더블클릭합니다.

```
review-bat.bat
```

<br>

## 단축키

|---|---|
| `Space` | 재생 / 정지 |
| `← / →` | 이전 / 다음 프레임 |
| `L` | 레이어 패널 열기/닫기 |
| `H` | 전체 보기 |
| `Z` | 줌 초기화 |
| `T` | 항상 위 토글 |
| `Delete` | 선택 삭제 |
| `Esc` | 선택 해제 |
| `Ctrl+Z` | 실행 취소 |
| `Ctrl+Shift+Z` | 다시 실행 |
| `Ctrl+A` | 전체 선택 |
| `Ctrl+C` | 복사 |
| `Ctrl+D` | 복제 |
| `Ctrl+V` | 붙여넣기 |
| `Ctrl+G` | 그룹 추가 |
| `Ctrl+T` | 텍스트 추가 |
| `Ctrl+I` | 색상 반전 |
| `Ctrl+N` | 새 프로젝트 |
| `Ctrl+O` | 열기 |
| `Ctrl+S` | 저장 |
| `Ctrl+Shift+S` | 다른 이름으로 저장 |
